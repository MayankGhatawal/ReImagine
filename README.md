## ğŸ› ï¸ Installation and Setup

1. **Clone the repository**  
   ```bash
   git clone https://github.com/MayankGhatawal/ReImagine.git
   cd ReImagine
   ```

2. **Install dependencies**  
   ```bash
   npm install
   ```

3. **Run the application**  
   ```bash

   ```

